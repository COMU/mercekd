![WTFPL](http://www.wtfpl.net/wp-content/uploads/2012/12/wtfpl-badge-1.png)

## INSTALLATION

    $ sudo apt-get install git
    $ git clone git://github.com/COMU/mercekd.git
    $ git checkout develop
    $ python bootstrap.py
    $ ./bin/buildout
    $ ./bin/django runserver
    $ ./bin/start_master.sh

That's it!
